RSS Blogroll Network

About

RSS feeds can programmatically define a blogroll using OPML. These blogrolls help people who read your blog discover other websites you think are worth promoting.

This project maps connections between blogs and other RSS-enabled entities. The size and interconnectedness of the network can be tracked over time.
